# Vaani — React + Vite starter

This repository is a minimal React app scaffolded with Vite and branded as **Vaani**.

Requirements:
- Node.js >= 20.19.0 (or 22.12+)

Local development:

1. Install dependencies:

```
npm install
```

2. Start the dev server:

```
npm run dev
```

If you see an error that Vite requires a newer Node.js version, upgrade Node (for example via `nvm install 20.19` or install Node 22+) and re-run `npm install`.

This starter includes ESLint and basic assets in `src/`. Edit `src/App.jsx` to customize the Vaani UI.
